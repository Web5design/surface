Surface Demo

The current features are:

  • Custom font for the surface!
  • Inserting text via the keyboard
  • Clicking within the text to change the cursor's' location
  • Deleting or insering text in the selected location
  • Inserting new lines via the enter key
  • Navigating by characters right and left with the arrows
  • Navigating by words right and left with the arrows using alt
  • Use Home and End to go to the beginning or end of the text
  • PageUp and PageDown to go to the beginning or end of the text
